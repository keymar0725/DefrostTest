# DefrostTest
解凍テスト自動生成プログラム


計測ソフトより生成したCSVファイルを読み込み、1時間ごとの温度を新しいListに追加し、それをプロットする

# Requirement

* python3.10.5

# Installation

```bash
pip install selenium
brew install chromedriver
```

# Usage

in home directory

```bash
git clone https://github.com/keymar0725/DefrostTest.git
cd (cloneしたディレクトリ)
python3 ~/exmaple/main.py
```
